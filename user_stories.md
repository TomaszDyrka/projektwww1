# User Stories

## Prezentacja cennika usług IT dla klienta
### User Story:

-   Jako: Właściciel małej firmy (potencjalny klient)

-   Chcę: zobaczyć przejrzysty podział usług wraz z cenami miesięcznymi i godzinowymi

-   Aby: szybko ocenić, czy oferta mieści się w moim budżecie przed podjęciem kontaktu.


## Formularz szybkiego kontaktu
### User Story:

-   Jako: Użytkownik odwiedzający stronę

-   Chcę: mieć możliwość wysłania wiadomości bezpośrednio ze strony "Kontakt"

-   Aby: nie musieć kopiować adresu e-mail i otwierać zewnętrznego klienta pocztowego.

## Znalezienie beneficjenta programu sponsoringowego
### User Story:

-   Jako: (Potencjalny) sponsor szukający beneficjenta

-   Chcę: znaleźć firmę której ideały zgadzają się z moimi

-   Aby: nawiązać lukratywną współpracę.
